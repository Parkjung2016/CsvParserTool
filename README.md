# PJDev Data Tool

Unity 프로젝트용 XLSX 데이터 작성, 검증, CSV 및 코드 생성을 지원하는 GUI/CLI 도구입니다.

## 미니게임 확장

Export 중 실행되는 게임은 Unity식 생명주기와 소스 자동 등록을 지원하며 모든 게임은 GUI EXE에 포함됩니다. 빠른 제작 방법은 [Mini Game Framework](docs/MiniGameFramework.md)를 확인하세요.
